Smoothers for Groups

When you define aesthetics in the ggplot() function, those mappings hold for every aspect of your plot.

Normality of Residuals

Technically, all along weâ€™ve been assuming that the residuals of our linear model are normally distributed.

In Lab 4 we made a density plot of the residuals from a linear model, which is exactly what we will do for a simple linear regression.

Recall that to obtain the residuals from the data model, we need to carry out the following steps:

  1. Fit a model using lm, with a quantitative response and a categorical explanatory
  2. Pull off the residuals from the model and store in a new variable in the dataset

Once weâ€™ve done these steps, we can create a density plot or histogram of the modelâ€™s residuals. Using this plot, we can assess the assumption of normality of residuals!

# Fit a linear model to the data 
evals_lm <- lm(score ~ bty_avg, data = evals)

evals %>% 
  # Create a new variable that's the residuals from the linear model 
  mutate(residuals = residuals(evals_lm)) %>% 
  # Plot a density plot of the residuals 
  ggplot(aes(x = residuals)) +
  # Choose your color! 
  geom_density(fill = "tomato")

# The standard base R plot for assessing normality of residuals  
plot(evals_lm, which = 2)
